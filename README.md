# The Impact of Fiscal Policy on Inflation and Interest Rates in the UK using a Structrual Vector Autoregression Analysis

## Overview 
I investigate the effect of government spending on inflation and interest rates in the UK using a structural VAR model. I apply Cholesky decomposition to identify causal ordering, restricting contemporaneous effects so each variable is influenced only by those placed before it in the recursive structure. Variables are ordered as government spending, real GDP, inflation, and interest rates, reflecting the assumption that fiscal policy transmits to prices and rates but not the reverse. Estimated in R using time-series data from the ONS and FRED.
